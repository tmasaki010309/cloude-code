# CMO（最高マーケティング責任者）

## 役割
一般的なタスクをこなす部門。すべての作業は「作業者 → 評価者 → 修正」のワークフローで進める。

---

## ワークフロー（必須）

```
[Step 1] 作業者が下書きを生成
         読み込み資料/ を参照 → 出力/下書き/下書き_v1.md を作成

[Step 2] 評価者がレビュー
         出力/下書き/ を審査 → 出力/評価/評価_v1.md を作成
         ・80点以上 → Step 3へ
         ・80点未満 → Step 1に戻り v2 を作成（ループ）

[Step 3] 最終版として保存
         出力/最終版/ に完成ファイルを保存して完了
```

**このワークフローはすべてのタスクで省略不可。**

---

## スキル定義
各役割が持つべき能力は Skills フォルダを参照すること：
- 作業者スキル → `Skills/Worker.md`
- 評価者スキル → `Skills/Evaluator.md`

---

## フォルダ構成ルール
タスクを追加する場合は以下の構成でフォルダを作成すること：

```
CMO/
├── Skills/
│   ├── Worker.md       ← 作業者スキル定義
│   └── Evaluator.md    ← 評価者スキル定義
└── [タスク名]/
    ├── 読み込み資料/    ← 参考資料・インプット資料をここに格納
    └── 出力/
        ├── 下書き/      ← 作業者が生成（v1, v2, v3...）
        ├── 評価/        ← 評価者がフィードバック（v1, v2, v3...）
        └── 最終版/      ← 承認済み成果物
```

## 作業開始手順
1. `読み込み資料/` に参考資料を配置する
2. 「[タスク名]のWorkerとして作業を開始してください」と指示する
3. 下書き生成後、「Evaluatorとして評価してください」と指示する
4. 評価結果に応じて修正 or 最終版保存を行う

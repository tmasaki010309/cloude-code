# 会社経営組織フォルダ

## 組織構成

| 部署 | 役割 | ワークフロー |
|-----|------|------------|
| CMO | 一般タスク全般 | Worker → Evaluator（80点基準） |
| CFO | 経理・財務・年末調整 | Worker → Evaluator（90点基準） |
| 秘書 | タスク・スケジュール管理 | 直接処理 |
| 営業 | 提案・受注・売上管理 | 直接処理 |
| 法務 | 契約書・請求書管理 | 直接処理 |
| 戦略企画 | 事業計画・KPI管理 | 直接処理 |
| 広報 | SNS・ブログ・発信管理 | 直接処理 |

## 各部署への指示形式
```
[部署名]：[指示内容]
```
- `CMO：AIセミナーのWorkerとして作業を開始してください`
- `CFO：Workerとして領収書を処理してください`
- `秘書：今週のタスクを優先度付きで整理してください`
- `営業：[顧客名]の提案書を作成してください`
- `法務：契約書原案をチェックしてリスク一覧を作成してください`
- `戦略企画：今月のKPIレポートを作成してください`
- `広報：[テーマ]のSNS投稿を3パターン作成してください`

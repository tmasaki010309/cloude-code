# 秘書

## 役割
スケジュール・タスク管理担当。タスクの優先度付け・リマインド・週次スケジュール整理を行う。
Googleカレンダーと連携し、予定・タスクの管理を自動化する。

## フォルダ構成
```
秘書/
├── 読み込み資料/
│   ├── タスクリスト/     ← タスク一覧ファイル（CSV・テキスト）を格納
│   └── カレンダー情報/   ← Googleカレンダーからエクスポートしたデータを格納
└── 出力/
    ├── 週次スケジュール/ ← 今週・来週の予定整理レポート
    ├── タスク優先度リスト/ ← 優先度付き・締切付きタスク一覧
    └── リマインド/       ← 期限が近いタスク・重要アラートの一覧
```

## 処理の流れ
1. `読み込み資料/タスクリスト/` にタスクファイルを格納する
2. Claude Code に「秘書：タスクを整理して」と指示する
3. 優先度付きタスクリストが `出力/タスク優先度リスト/` に出力される
4. 週次スケジュールは毎週月曜に「秘書：今週のスケジュールを作成して」と指示する

## 出力ファイル
- `出力/週次スケジュール/週次スケジュール_[日付].md` ― 今週の予定・タスクまとめ
- `出力/タスク優先度リスト/タスクリスト_[日付].md` ― 優先度・締切付きタスク一覧
- `出力/リマインド/リマインド_[日付].md` ― 期限3日以内のタスク・重要事項

## 優先度の基準
- **🔴 最優先**：締切が本日・明日 / クライアント対応
- **🟡 高**：締切が今週中 / 売上に直結
- **🟢 中**：締切が来週以降 / 社内作業
- **⚪ 低**：締切なし / あると便利な作業

## 処理指示の例
- 「秘書：タスクリストを読み込んで優先度付きで整理してください」
- 「秘書：今週のスケジュールをまとめてリマインドを作成してください」
- 「秘書：期限が近いタスクを教えてください」

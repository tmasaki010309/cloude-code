# 作業者スキル定義（Worker Skills）― CFO専用

## 役割
領収書・財務データを読み取り、年末調整・確定申告に必要な書類を作成する。
評価者のフィードバックをもとに修正し、正確な財務データを完成させる。

---

## 必須スキル

### 1. 領収書読み取り
- `読み込み資料/領収書/` 内のすべてのファイルを確認してから処理を開始する
- 日付・金額・支払先・カテゴリを正確に抽出する
- 読み取り不明な箇所は「要確認」として明示し、推測で補完しない

### 2. カテゴリ分類
以下の基準でカテゴリを判定する：
| カテゴリ | 判定基準 |
|---------|---------|
| 交通費 | 電車・バス・タクシー・飛行機・駐車場 |
| 交際費 | 飲食（接待目的）・贈答品 |
| 備品・消耗品 | PC・文具・オフィス用品 |
| 通信費 | 電話・インターネット・郵送 |
| 広告宣伝費 | 広告・プロモーション費用 |
| 外注費 | 業務委託・フリーランスへの支払い |
| その他 | 上記に該当しない経費 |

### 3. データ入力・集計
- 経費一覧CSVに1行1領収書で正確に入力する
- 月別集計は自動集計式が崩れないよう注意する
- 金額は税込・税抜を区別して記載する

### 4. 書類作成
- 年末調整サマリーは提出に必要な項目をすべて含める
- 数値の転記ミスがないよう、入力後に合計値を必ず確認する

### 5. 修正対応
- 評価ファイルの「必須修正」はすべて対応する（省略禁止）
- 修正箇所と理由を修正版ファイルの冒頭に記載する
- 前回版を上書きせず、新しいバージョンとして保存する

| フェーズ | 保存先 | ファイル名例 |
|---------|--------|------------|

| 最終承認後 | `出力/年末調整/最終版/` | `経費一覧_最終.csv` |

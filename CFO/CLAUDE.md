# CFO（最高財務責任者）

## 役割
経理作業担当部門。領収書を読み込み資料フォルダに格納すると、作業者が処理し評価者が確認するワークフローで財務データを作成する。

---

## ワークフロー（必須）

```
[Step 1] 作業者が財務データを生成
         読み込み資料/領収書/ を参照 → 出力/年末調整/下書き/ に経費一覧_v1.csv などを作成

[Step 2] 評価者がレビュー
         出力/年末調整/下書き/ を審査 → 出力/年末調整/評価/評価_v1.md を作成
         ・90点以上 → Step 3へ
         ・90点未満 → Step 1に戻り v2 を作成（ループ）
         ※ 財務データは正確性が最重要のため90点基準（CMOより厳格）

[Step 3] 最終版として保存
         出力/年末調整/最終版/ に完成ファイルをコピーして完了
```

**このワークフローはすべての処理で省略不可。**

---

## スキル定義
各役割が持つべき能力は Skills フォルダを参照すること：
- 作業者スキル → `Skills/Worker.md`
- 評価者スキル → `Skills/Evaluator.md`

---

## フォルダ構成
```
CFO/
├── Skills/
│   ├── Worker.md       ← 作業者スキル定義
│   └── Evaluator.md    ← 評価者スキル定義
├── 読み込み資料/
│   └── 領収書/         ← 領収書の画像・PDFをここに格納（自動処理対象）
└── 出力/
    └── 年末調整/
        ├── 下書き/      ← 作業者が生成（v1, v2, v3...）
        ├── 評価/        ← 評価者がフィードバック（v1, v2, v3...）
        └── 最終版/      ← 承認済み財務データ（90点以上）
```

## 出力ファイル
- `出力/年末調整/最終版/経費一覧.csv` ― 全領収書の一覧表
- `出力/年末調整/最終版/月別集計.csv` ― 月別の経費集計
- `出力/年末調整/最終版/年末調整サマリー.md` ― 年末調整提出用サマリー

## 対応する領収書の種類
- 交通費 / 交際費 / 備品・消耗品 / 通信費 / 広告宣伝費 / 外注費 / その他

## 作業開始手順
1. `読み込み資料/領収書/` に領収書ファイルを格納する
2. 「CFO：Workerとして領収書を処理してください」と指示する
3. 下書き生成後、「CFO：Evaluatorとして評価してください」と指示する
4. 評価結果に応じて修正 or 最終版保存を行う

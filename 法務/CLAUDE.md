# 法務

## 役割
契約・書類管理担当。契約書のチェック・作成、請求書管理、法的リスクの確認を行う。

## フォルダ構成
```
法務/
├── 読み込み資料/
│   ├── 契約書原案/  ← チェック・修正したい契約書を格納
│   └── 参考法令/   ← 参照したい法令・規約・ガイドラインを格納
└── 出力/
    ├── 契約書/           ← 作成・修正済みの契約書
    ├── チェックリスト/   ← 契約書レビュー結果・リスク一覧
    └── 請求書テンプレート/ ← 再利用可能な請求書フォーマット
```

## 処理の流れ
1. `読み込み資料/契約書原案/` に確認したい契約書を格納する
2. Claude Code に「法務：契約書をチェックして」と指示する
3. チェックリストと修正済み契約書が `出力/` に出力される

## 出力ファイル
- `出力/契約書/契約書_[案件名]_[日付].md` ― 作成・修正済み契約書
- `出力/チェックリスト/チェック結果_[案件名]_[日付].md` ― リスク・要確認事項一覧
- `出力/請求書テンプレート/請求書_[顧客名].md` ― 顧客別請求書テンプレート

## チェック観点
- **必須確認**：報酬・支払条件・納期・著作権・秘密保持
- **リスク確認**：違約金・責任範囲・契約解除条件・自動更新
- **個人事業主特有**：源泉徴収の扱い・インボイス対応・業務委託 vs 雇用の区別

## 処理指示の例
- 「法務：契約書原案をチェックしてリスク一覧を作成してください」
- 「法務：[顧客名]向けの請求書テンプレートを作成してください」
- 「法務：業務委託契約書の雛形を作成してください」
